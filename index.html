HTML

  • HTML or HyperText Markup Language is madeup of elements that provide basic content and structure to a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Child elements are elements nested inside other elements. A good example is the bullet points you're currently reading, which are each child elements to a ul "Unordered List" element, which is itself a child element to the body element, or the "Parent Element."

CSS

  • A selector is assigned to the following property and the value applied to it.
  • An example of a selector is the img selector, where we can then write rules we want for a linked image.
  • An example of a property is the height property, which has both the img selector assigned to it linking it to the image, and the 50 px value applied to it making the image 50 pixels in height.
  • A margin indicates how much space we want around the outside of an element, while a padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • In git, "staging" is the process of preparing your code to upload from the local to the remote hub, while "committing" is the process of uploading it to the remote hub.
  • "GitHub", the internet hosting service for git, was created in 2008.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function MUST be called in your code(after it's defined/declared) to execute.
  • Javascript has several built in objects like Math that can give you access to useful functions like Math.random().
